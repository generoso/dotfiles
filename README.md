# README

This is my `.dotfiles` project.

Your dotfiles define how you personalize your system.


## Getting started

```
git clone https://github.com/generoso/dotfiles.git
mv dotfiles $HOME/.dotfiles
cd $HOME/.dotfiles
./install.sh
```
